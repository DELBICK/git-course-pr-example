```
Жили у бабуси  
Два весёлых гуся:  
Один — серый,  
Другой — белый —  
Два весёлых гуся.    

Вытянули шеи —  
У кого длиннее?  
Один — серый,  
Другой — белый —  
У кого длиннее?    

Мыли гуси лапки  
В луже у канавки.  
Один — серый,  
Другой — белый —  
Спрятались в канавке.    

Вот кричит бабуся:  
Ой, пропали гуси!  
Один — серый,  
Другой — белый —  
Гуси мои, гуси.    

Выходили гуси
Кланялись бабусе  
Один - серый, 
другой - белый 
Кланялись бабусе.
```
